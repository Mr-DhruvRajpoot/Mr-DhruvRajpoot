## Featured Projects

### 🔁 [Activation Optimisation — A/B Analysis](https://github.com/Mr-DhruvRajpoot/bank-campaign-activation-analysis)

**The Fix:** Chi-square testing validated that users spending >5 min on KYC converted at 4x the rate. I guided a friction reduction flow resulting in a **+14pp lift**.
`Python` `scipy` `Funnel Analysis`
